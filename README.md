# Travel Planner

這是一個可上傳到 GitHub 並部署到 Vercel 的 React + Vite 旅遊協作網站範例。

## 本機啟動

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```

## 部署到 Vercel

1. 把整個資料夾上傳到 GitHub
2. 登入 Vercel
3. Import Git Repository
4. 選擇這個 repository
5. Deploy

## 備註

- Google Calendar 同步需要你自己填入 Google OAuth Client ID
- 目前資料儲存在瀏覽器 localStorage
- 分享連結與協作功能是前端示意版，不是跨裝置即時同步版
